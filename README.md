# Home Lab - Infrastructure & Security Monitoring

This repository documents my home laboratory environment for hands-on practice with IT systems, moniroting, and security detections.

## Purpose

A sandbox to learn and experiment with:

- Windows and Linux administration
- Network configuration and troubleshooting
- Automation (PowerShell, Bash)
- Monitoring and alerting (Wazuh)
- Security telemetry and detections (Sysmon, auditd)

The lab evolves as I explore new topics - from basic IT tasks to security use cases.

## Structure

- `docs/` - Technical documentation (network, system configs, troubleshooting notes)
- `scripts/` - Automation scripts (PowerShell, Bash)
- `monitoring/` - Monitoring configurations (Wazuh rules, dashboards)

## Highlights

- **Isolated lab network** - Physical segment with bridging and Wi-Fi for internet access
- **Endpoint telemetry** - Sysmon (Windows) and auditd (Linux) forwarding to Wazuh
- **Troubleshooting docs** - Real issues solved in the lab (agent offline, performance, connectivity)

## Skills demonstrated

- Windows / Linux system administration
- Networking (subnets, routing, troubleshooting)
- Monitoring & SIEM (Wazuh)
- Scripting (Powershell, Bash)
- Technical documentation
